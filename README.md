# CodePDE-Agent | 基于 LLM 的 1D 反应扩散方程自动求解器进化框架

针对偏微分方程（PDE）数值求解中，传统人工编写高精度算子耗时且易出错的问题，本项目开发了一套基于 LLM 的**闭环自动迭代演化框架**。该框架能够针对给定的物理方程（如 Reaction-Diffusion Equation）自动生成、测试并演化高性能的数值解法代码 。

## 🌟 核心特性

* 
**双闭环进化机制**：设计了“生成-执行-评估-反馈”的循环系统。利用 LLM 生成基于 **Strang Splitting（对称分步法）** 和 **Spectral Methods（谱方法）** 的求解器 。


* 
**自动化数值诊断**：自研评估器（Evaluator），不仅检测绝对误差（nRMSE），还通过多尺度时间步长（$dt$）测试自动计算 **数值收敛阶（Convergence Rate）**，作为代码数学正确性的硬性判据 。


* 
**智能反馈注入逻辑**：构建了分级反馈机制（Multi-level Feedback），根据计算结果自动判定“物理不稳定性”、“次优收敛”或“参数匹配”等问题，并动态注入数学诊断指令引导模型自主修复 。



## 🛠️ 技术栈

* 
**编程语言**：Python (NumPy, PyFFT) 


* 
**配置管理**：Hydra 


* 
**模型接口**：DeepSeek / Claude API 


* 
**数学核心**：偏微分方程数值解、数值分析（收敛性理论） 



## 📈 技术突破与成果

1. 
**算法逻辑自动化**：成功引导 LLM 实现二阶对称分裂算法，在多个样本测试中达到理论 **二阶时间收敛精度（Average Rate $\approx 1.96$）**，证明了 Agent 对复杂数学逻辑的精准掌握 。


2. 
**数值稳定性控制**：针对非线性反应项产生的 Gibbs 震荡，通过反馈机制强制 Agent 引入 **谱截断去混叠（2/3 De-aliasing Rule）** 与物理值域约束（Clipping），显著提升了长程演化的稳定性 。


3. **Code-as-Policy 应用**：通过构建自动纠错环路，有效解决了 LLM 在生成高性能科学计算代码时常出现的“幻觉”问题（如忽略物理量纲、步长缩放错误等）。

## 🏗️ 系统架构

1. **Generator**: 接收 PDE 描述，生成基于谱方法的离散化代码。
2. **Evaluator**: 执行数值仿真，进行收敛阶测试与稳定性分析。
3. **Diagnoser**: 对比理论解（或高精度参考解），生成分级修复建议。
4. **Evolver**: 根据反馈调整 Prompt，驱动代码版本进化。
